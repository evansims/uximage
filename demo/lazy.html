This is a demo of uxImage's lazy loading functionality. 3 image assets are repeated 4 times. As you scroll down the page, these assets are loaded as needed. Once you've scrolled far enough beyond the images, they are unloaded to conserve memory and preserve page performance.

This technique is especially useful for mobile users where memory is more limited, and on websites that apply infinite scrolling methods where an unknown number of image assets might be in play.
